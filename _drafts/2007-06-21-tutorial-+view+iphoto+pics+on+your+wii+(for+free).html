I know the Wii has a built in photo viewer, but this is a nice option if you don't want to keep the pictures in multiple locations. Unfortunately, this one is Mac only guys....sorry Windows and Linux peeps

Things you'll need:
  • Nintendo Wii
  • iPhoto Library
  • iPhoto Access
Let's get started:
  1. Make sure you have the latest version of iPhoto installed on your computer.
  2. Make sure you have the latest version of the Opera browser installed on your Wii.
  3. Download and install iPhoto Access.
After you install iPhoto Access:
  1. First you want to make sure that your firewall has "Personal Web Sharing" enabled. To do so, open up system preferences>sharing and under the services tab check the box next to "Personal Web sharing."


  2. This next step is optional, but I recommend it. Since the Wii is not very good at display small text I went ahead and modified the CSS file that the program uses and simply increased the font size for easier readability on the Wii. The new CSS file can be downloaded here. Once it's downloaded simply drag and drop it to:
    Users/your user name/Sites/wipha/skin/orig/wipha.css
    When it asks if it's ok to replace the existing file say yes.
  3. It should bring you to the iPhoto Access homepage which will load in your default browser. Go ahead and bookmark this page so you can easily access it in the future. If it doesn't load for some reason, or you close the window by accident the default address is: http://your local ip address/~your username/wipha/main.php
    So mine looked like this: http://127.0.0.1/~Jenny/wipha/main.php


  5. It's now time to login for the first time and configure your albums. The default username and password are admin/admin
  6. Once your logged in as the admin, click the "admin" button in the upper right hand corner of the browser window to begin configuring your photos.



  7. Adding a new photo library is pretty simple. Under the "Libraries" section enter whatever you like into the Name field, I chose "My Pics," and under the enter in the location of your iPhoto library. The default iPhoto library path is:
    /Users/your user/Pictures/iPhoto Library

  8. After you click the add button, if all went well you are finished. At this point you can add more users and define their permissions.
  9. To browse your new library logout and then login back in. Select the library you want to browse from the drop down menu and the number of pictures you want to view and click the display button. You can also go back to the portal page at any point by clicking the "New Search" button.


Configuring the Wii:
  1. Load the "Internet Channel"
  2. Click on the "WWW" icon
  3. Enter in the IP address you wrote down in step 1 of "After You Install iPhoto Access " into the address bar. In my case it was "http://127.0.0.1/~Jenny/wipha/main.php
  4. Enter the username and password for the user you created earlier and click the log in button or just use admin/admin. You should see a portal page. Congratulations you are now streaming your iPhoto library to your Wii!!!
*I also saved the ip address "http://127.0.0.1/~Jenny/wipha/main.php" to my favorites and renamed it "My Photos" for easier access.
