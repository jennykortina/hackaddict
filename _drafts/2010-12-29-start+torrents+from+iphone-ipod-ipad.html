--- layout: post title: Start Torrents from iPhone/iPod/iPad date: '2010-12-29T13:17:00.003-05:00' author: jenny tags: modified_time: '2010-12-29T13:39:41.129-05:00' blogger_id: tag:blogger.com,1999:blog-5518298822864690168.post-2699110375022603424 --- I've always wanted to be able to start torrents from my iDevice (iPhone,iPod,iPad) and finally figured it out. Below I describe how to start the torrent from your iDevice and once it's completed have it auto-add to iTunes or move to a movie folder so it can be streamed back to your iDevice immediately.

Softwares we're going to need:
iPhone/iPad/iPod:
iCab Mobile - Safari replacement that let's you download files and save them to Dropbox
Zumocast - free software to stream music/movies from our mac

Mac:
Transmission - bitorrent software
Dropbox - file syncing software used to start torrents
Hazel - file organizer used to move files to correct place after downloaded

Initial Set up of Mac:
  1. If you don't already have it, sign up for a free dropbox account and then download and install the software
  2. Download and install transmission
  3. Create two folders in your Home>Downloads folder of your mac: Incomplete Torrents and Complete Torrents
  4. Create a third folder inside your Dropbox folder called "Torrents"
  5. Fire up transmission
  6. Transmission> Preferences >Transfers
    • Under "Settings: Default Locations" select our "Completed Torrents" folder we created in step 3
    • Check "Settings: Keep incomplete files in" and select our "Incomplete Torrents" folder we created in step 3
    • Check" Auto add: Watch for torrent files in" and select our "Torrents" folder we created in step 4
  8. Transmission is now configured
Setting up iDevice:
  1. Download and install iCab
  2. Click settings>Dropbox
  3. Click "Link Dropbox" and enter in your Dropbox credentials
  4. iCab Mobile will create a self named folder in your Dropbox account
Finalizing Mac:
  1. Download and install hazel
  2. Open up system preferences>Other>Hazel
  3. On the left hand side under folders click the plus button. Add "iCab Mobile" folder that lives inside your Dropbox folder
  4. Under Rules click the plus button to add a new rule
  5. On the left hand side under folders click the plus button. Add "Compeled Torrents" folder that lives inside your downloads folder
  6. Under Rules click the plus button to add a new rule
