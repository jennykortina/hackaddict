--- layout: post title: date: '2011-10-23T18:33:00.000-04:00' author: jenny tags: modified_time: '2011-10-23T18:33:43.653-04:00' blogger_id: tag:blogger.com,1999:blog-5518298822864690168.post-4763131065669802548 --- A while back I read about Stephen Myers webcam-enabled dog-treat dispenser.  I was impressed so along with my friend Shreyans we made our own version that we can treat my dog, Chewie, with and so can you!

Here's how it works:
Stop by www.treatchewie.com.  Pay twenty five cents or more and your name will be added to the Chewie treat queue.  When it's time for your treat dispensed you'll receive a Facetime call (you must have a Facetime enabled device to receive the call) where you'll get to see your treat drop and Chewie enjoy it!

Parts list:
  • 1 x arduino
  • 1 x USB cable for arduino
  • 1 x continuous rotation servo
  • 1 x gumball machine
  • 1 x webcam
Software list:
  • Servo libray -- github
  • Control Facetime via Python
  • 1 x web server (I used a mac mini I have running in my house)
  • Venmo API
  • Mongo DB
